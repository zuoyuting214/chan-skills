# `chanjing-tts-voice-clone` 速用版

## 什么时候用

- 想让声音像某个人
- 想长期复用同一个声音
- 比起“能听”，你更在意“辨识度”

## 直接怎么说

- “我想复刻品牌主理人的声音，保留温和、可信的感觉，用来做课程讲解。”
- “参考这个声音做带货口播，要保留人物辨识度，但表达更利落一点。”

## 跟 OpenClaw 这样说，更容易触发

- 直接说“复刻声音”“克隆声音”“像某个人的声音”。
- 最好带上“保留什么感觉”，比如温和、成熟、可信、亲切。
- 如果你已经有素材，可以直接说“参考这个声音”。

高概率说法：

- “请参考这个声音，复刻成可用来做课程讲解的版本。”
- “我想做一个像品牌主理人的声音，重点保留温和和可信感。”

## 不要这么说

“帮我克隆这个声音然后读文案。”

## 这么说更好

“这是文案内容：xxx，这是参考音频：xxx。重点有成熟、可信、像老师讲话的感觉。这段内容用于课程讲解，不要太像广告，帮我文案润色一下，复刻我的声音，合成音频”
