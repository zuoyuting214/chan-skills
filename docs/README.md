# chan-skills 速用指南

这套文档只解决一件事：不会编程的用户，怎么在对话框里一句话把需求说清楚。

## 先记住 4 条

1. 先说你要做什么，不要只说“帮我生成一个”。
2. 说清用途和人群，不要只说“高级一点”“好看一点”。
3. 说清你最在意什么，比如真实感、转化感、品牌统一。
4. 先要 1 个版本确认，再批量做。

## 通用模板


> 我想做【内容类型】，用于【业务场景】，给【目标人群】看。  
> 我最在意【真实感 / 转化感 / 品牌统一 / 创意感】。  
> 风格希望是【风格描述】。  
> 请先给我【1个版本 / 3个方向】。

## 跟 OpenClaw 这样说，更容易触发 skill

1. 直接说条件和目标
2. 尽量带上能力关键词，比如“口播”“数字人”“复刻声音”“文本驱动唇形”“语音驱动唇形”“对口型”“修改口型”“视频翻译”“海报”。
3. 如果你已经知道要哪种能力，可以直接点名说“请用蝉镜数字人技能”“请用chanjing-tts skill”“请用对口型skill”“请用视频翻译技能”。
4. 先说“先出 1 个版本”，比一上来就说“批量做 50 个”更容易执行对。


## 怎么选 skill

- 文字变声音：`chanjing-tts`
- 声音像某个人：`chanjing-tts-voice-clone`
- 文本驱动唇形 / 语音驱动唇形 / 对口型 / 修改口型 / 视频翻译：`chanjing-avatar`
- 标准数字人口播：`chanjing-video-compose`
- 训练长期复用数字人：`chanjing-customised-person`
- 从人设生成数字人形象/视频：`chanjing-text-to-digital-person`
- 做海报、创意图、创意视频：`chanjing-ai-creation`
- 先把蝉镜账号搞定：`chanjing-credentials-guard`
