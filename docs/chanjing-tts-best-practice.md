# `chanjing-tts` 速用版

## 什么时候用

- 想把文案变成口播声音
- 想先做声音，再决定要不要做视频
- 不要求像某个真人

## 直接怎么说

- “帮我做一段护肤带货口播，面向 25-35 岁女性，语气专业、亲切，别太像叫卖。”
- “把这段课程文案做成老师讲解感的音频，温和一点。”

## 跟 OpenClaw 这样说，更容易触发

- 直接说“口播”“配音”“把文案变成声音”。
- 最好带上用途词，比如“带货口播”“课程讲解音频”“品牌旁白”。
- 如果不想复刻真人声音，可以直接说“不要像某个真人，只要合适口播”。

高概率说法：

- “请把这段文案做成护肤带货口播。”
- “请把这段内容变成课程讲解音频，语气像老师上课。”

## 不要这么说

“帮我做个口播音频。”

## 这么说更好

“这是我的护肤品文案：xxx；帮我做一段 30 秒带货口播，给第一次了解产品的人听。语气亲切、专业，不要太推销。文案太书面的话先改成口语。”
