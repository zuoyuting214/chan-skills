# `chanjing-credentials-guard` 速用版

## 什么时候用

- 第一次使用任何 chanjing 相关 skill
- 你不确定蝉镜账号有没有配置好
- 你怀疑 AK/SK 或 Token 状态有问题

## 直接怎么说

- “请先帮我检查蝉镜账号现在能不能正常用。”
- “我第一次用蝉镜，先带我把 AK/SK 配好。”
- “先帮我确认凭证状态，再继续做后面的内容。”

## 跟 OpenClaw 这样说，更容易触发

- 直接说“先检查蝉镜账号”“先配置 AK/SK”“先确认凭证状态”。
- 如果你是第一次使用，最好明确说“先不要生成内容，先把账号配置好”。
- 如果你是接着做别的任务，可以直接说“先检查凭证，再继续做 TTS / 数字人 / 对口型”。

高概率说法：

- “我第一次用蝉镜，请先带我完成账号配置。”
- “请先检查我的蝉镜凭证状态，没问题再继续。”

## 不要这么说

“帮我开通一下。”

## 这么说更好

“我第一次用蝉镜，先帮我完成 AK/SK 配置，并确认当前状态可以继续调用后面的 skill。配置好后再告诉我下一步该怎么做。”
