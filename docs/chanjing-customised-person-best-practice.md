# `chanjing-customised-person` 速用版

## 什么时候用

- 想把一个真人做成长期复用数字人
- 以后很多内容都想让同一个角色出镜
- 你做的是品牌 IP、讲师数字化、主播分身

## 直接怎么说

- “我想把这个真人做成长期复用的数字讲师，后续用于课程讲解。”
- “我想把品牌主理人做成固定数字角色，后面持续做产品介绍。”

## 跟 OpenClaw 这样说，更容易触发

- 直接说“做成长期复用数字人”“训练一个数字人角色”“固定数字角色”。
- 最好带上“以后长期使用”“后续持续出镜”这类词。
- 不要只说“做成数字人”，否则容易和单次出片混淆。

高概率说法：

- “请把这个真人训练成一个长期复用的数字人角色。”
- “我想做一个以后持续出镜的固定数字角色。”

## 不要这么说

“帮我把这个人做成数字人。”

## 这么说更好

“这是我的真人素材。请帮我把这个真人做成长期复用的数字讲师，后续用于课程讲解和知识短视频，风格要可信、专业。”
