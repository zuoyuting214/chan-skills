# `chanjing-text-to-digital-person` 速用版

## 什么时候用

- 手里没有真人素材
- 想先试数字人人设
- 想先看形象，再决定要不要做视频

## 直接怎么说

- “帮我设计一个知识型女性数字人，用于母婴内容，给新手妈妈看，气质温柔、专业、有安全感。先出几个形象方向。”
- “我想做一个年轻男性科技博主形象，适合讲 AI 工具，先看图再决定视频。”

## 跟 OpenClaw 这样说，更容易触发

- 直接说“设计一个数字人形象”“先出人设方向”“先看图再做视频”。
- 最好带上角色身份词，比如“讲师”“博主”“顾问”“主播”。
- 如果你还没想做视频，可以直接说“先只出形象方案”。

高概率说法：

- “请先帮我设计一个数字人形象，给我 3 个方向。”
- “我想先做一个知识博主人设，先看图，再决定要不要做视频。”

## 不要这么说

“帮我做一个数字人形象。”

## 这么说更好

“帮我设计一个做职场成长内容的女性数字人，面向刚毕业的年轻白领。形象要有亲和力，也要显得可靠。先只出形象方案，给我 3 个方向。”
