# `chanjing-ai-creation` 速用版

## 什么时候用

- 想做海报、封面、创意图、创意视频
- 想先试几个方向
- 更在意创意感和画面感

## 直接怎么说

- “帮我做一张护肤品牌海报，用于电商首图，第一眼要干净、高级、可信。”
- “帮我做一个 AI 工具推广短视频概念画面，重点是未来感和效率感。”

## 跟 OpenClaw 这样说，更容易触发

- 直接说“海报”“封面图”“创意图”“创意视频”“概念画面”。
- 最好带上用途词，比如“电商首图”“投流素材”“品牌海报”。
- 如果你想先探索方向，可以直接说“先给我 3 个不同方向”。

高概率说法：

- “请帮我做一张电商首图海报。”
- “请先给我 3 个品牌海报方向，我来选一个。”

## 不要这么说

“帮我做一张海报。”

## 这么说更好

“请帮我做一张护肤产品电商海报，用于首页首图。第一眼要显得清透、安心、适合敏感肌，不要太像夸张广告。先给我 3 个方向，我选一个。”
