# `chanjing-avatar` 速用版

## 什么时候用

- 想做文本驱动唇形
- 想做语音驱动唇形
- 想做对口型、修改口型、视频翻译
- 已经有现成视频，不想重拍

## 直接怎么说

- “请帮我做文本驱动唇形，把这条视频里的口型按新的文案对齐。”
- “请帮我做语音驱动唇形，用新的配音去驱动这条视频的口型。”
- “请使用我提供的音频，帮我把这条视频对口型，尽量自然。”
- “请帮我做视频翻译，换成我提供的英文文案并把口型一起改掉。”

## 跟 OpenClaw 这样说，更容易触发

- 直接说“文本驱动唇形”“语音驱动唇形”“对口型”“修改口型”“视频翻译”。
- 最好说清是“按新文案改口型”还是“按新语音改口型”。
- 如果是翻译场景，直接说“视频翻译，并同步改口型”。
- 最好明确“保留原视频画面”“不要重拍”“不要重做成数字人”。

高概率说法：

- “请对这条视频做文本驱动唇形，按新的中文文案修改口型。”
- “请对这条视频做语音驱动唇形，用新的配音去对口型。”
- “请把这条视频翻译成英文，并把口型同步改成英文版本。”

## 不要这么说

“帮我改一下这个视频。”

## 这么说更好

“我想保留这条真人视频的镜头和人物状态，请按新的促销文案做文本驱动唇形。文案先润色得更口语一点，尽量让对口型看起来自然。”
