# chan-skills

Chan Openclaw skills，providing pratical tools with a focus on E-Commerce content creating.

蝉系Openclaw技能库，聚焦电商内容创作，提供实用的AI工具及技能包。

## Install 

```
# List available Chan skills
# 查看蝉系可安装skill列表
npx skills add chanjing-ai/chan-skills --list

# Install all Chan skills
# 安装蝉系所有skill 
npx skills add chanjing-ai/chan-skills

# Install certain Chan skill name
# 安装指定的skill
npx skills add chanjing-ai/chan-skills --skill chanjing-tts -y
```

## Available Skill 

| Name | Description |
|------|------|
| chanjing-tts | Bilingual text to speech using one of our provided voices. 根据指定的文本和参考音色，合成音频，支持中英文 ｜
| chanjing-tts-voice-clone | Bilingual text to speech using a voice you provide. 使用你提供的音色合成音频，支持中英文 ｜
